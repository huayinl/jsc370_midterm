# JSC370 Midterm

This repository contains my JSC370 midterm report, Evolution of Model Efficiency in the Open Source Landscape.

## Files

- `jsc370_midterm.html`: rendered report, ready to open in a browser
- `jsc370_midterm.qmd`: Quarto source file used to generate the report

## How to Read the Report
Open `jsc370_midterm.html` in any web browser.

## How to Run the Report

This report was written in Quarto with a Python Jupyter kernel.

Requirements:
- Quarto
- Python 3
- Jupyter
- Python packages used in the report, including `pandas`, `requests`, `plotly`, and `IPython`

You can install the required packages with
```bash
pip install pandas requests plotly numpy ipython jupyter ipykernel
```

To render the report from the source file, run:

```bash
quarto preview jsc370_midterm.qmd
```
